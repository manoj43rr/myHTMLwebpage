html Tag

The "html" tag is the basic tag for webpage the html tag determines the start of the webpage. The "html" tag has two major sections they are :

  1. Head Section
  2. Body Section

Head Section

The "Head" section is represented by Head Tag. The head tag contains that are not seen on the webpage but are essential for the webpage's details.

Some of the important tags that are actually used within the "head tag".

    • "title tag"

      • Within the title tag we feed the title of the webpage.

      You can change the title of this webpage by entering the name you want


    • "meta tag"

      • The tag in HTML is used to provide metadata about the HTML document. Metadata is data that describes other data, and in this context, it provides information about the web page, such as its character set, author, description, keywords, viewport settings, and more. The tag is placed within the section of an HTML document.

Body Section

The "Body" section contains the data ie., displayed on the webpage. The body tag contains all the elements that the user needs to display whether it could be images, video or audio etc. Everything you are seeing on this webpage is written within "Body tag".

Audio Tag

The audio tag is used to load the audio data on to the webpage. we have few attributes that makes the audio data to be handled by the user.

here are few of the attributes of the audio tag :

  • Controls : This attribute is a boolean attribute that is able to provide the controls that are required for user. Some of the controls are Play/Pause, mute/unmute, the line displaying the stage of audio.
  • autoplay : This attribute is also a boolean attribute that plays the audio data automatically when the webpage is loaded. It can be paused by the user.
  • loop : This is also a boolean attribute that is able to repeatedly play the audio data when the audio data is finished.
  • muted : This is also a boolean attribute that is used to mute the audio data initially but this can be unmuted by the user.
